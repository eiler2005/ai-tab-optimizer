# Screenshots

Place UI screenshots here. They are referenced in the root `README.md`.

## Required files

| File | View | How to capture |
|---|---|---|
| `ai-analysis.png` | AI tab → Analysis section (while analysis is running or after completion) | Open side panel, run AI analysis, screenshot the recommendations list |
| `topic-clusters.png` | AI tab → Analytics → Theme clusters (expanded) | After analysis, open Analytics tab, expand a cluster |
| `analytics.png` | AI tab → Analytics → full dashboard | Scroll through analytics: habits score, heatmap, insights |
| `tab-list.png` | Tabs view with a loaded session | Open side panel with 20+ tabs open, screenshot the tab list |
| `cleanup-session.png` | Cleanup Session in progress | Start a cleanup session, screenshot mid-flow |
| `chat-search.png` | Search tab with a query and results | Open Search tab, type a query, screenshot the results |

## Recommended specs

- Width: 400px (extension side panel width)
- Format: PNG
- Zoom: 100% (no browser zoom)
- Theme: match the extension default (light or dark — pick one consistently)

## Naming convention

Use lowercase, hyphenated names matching exactly the filenames in the table above.
